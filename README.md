### 前言

    为了降低后续同学学习内核的难度，我把《LDD3》中的代码和内容做了整理。

    2.删繁就简，将书中复杂的示例替换为简单示例，便于理解。

    3.总结概括，记录了每一章的主要内容，实验截图。
